ビット演算の挙動

ビット演算のオペランドは暗黙的に 32-bit 値に変換されます。 この性質を用いると、数値を 32-bit 整数に変換できます。

小数から整数への変換はもちろん、 32-bit 範囲に収まらない整数から 32-bit 整数への変換もできます。

// num を符号付き 32-bit 整数に変換する
function asI32(num) { return num >> 0; }

// num を符号なし 32-bit 整数に変換する
function asU32(num) { return num >>> 0; }

console.log(asU32(-1))         // 4294967295
console.log(asU32(4294967297)) // 1
console.log(asI32(4294967295)) // -1

ブロックの脱出

a: {
  console.log('これは表示されます');
  break a;
  console.log('これは表示されません');
};
console.log('これも表示されます');

export / import

変数や関数などの宣言に export キーワードをつけることで、それらを外部に公開することができます。

export const hello = "hello";
export function hoge() { ... }
export class Class { ... }

export されたものを使うには、 import します。

import { hello, hoge, Class } from './hoge.js';
