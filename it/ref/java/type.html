Java の型

整数・小数

型対応するRustの型
bytei8
charu16
shorti16
inti32
longu64
floatf32
doublef64

整数のリテラルは int 型、小数のリテラルは double 型とみなされます。

暗黙キャスト

拡大方向には暗黙キャストできますが、縮小方向にはできません。

byte ─┬─▶ char ──┬─▶ int ───▶ long ───▶ float ───▶ double
      │          │
      └─▶ short ─┘

ただし、代入の右辺が int 型の定数式の場合は例外です。 値が左辺の型の範囲に収まれば、暗黙的に縮小変換されます。

final int x = 16;
char a = (1 << x) - 1; // 65535

収まらなければコンパイルエラーになります。

final int x = 16;
// エラー
// char b = 1 << x;

Autoboxing, Autounboxing

整数型と対応する参照型とは、相互に暗黙変換されます。

final var l = new ArrayList<Object>();

// Boxing
l.add(10);

// class java.lang.Integer
System.out.println(l.get(0).getClass());

// Unboxing
final int i = (Integer)l.get(0);

汎整数拡張

int より小さな整数型は、式中で int に拡張されます。

char max = 65535;
char a = 0, b = 0;

a -= 1;
System.out.println(a == max); // true
System.out.println(b - 1 == max); // false

int より大きな型には拡張されません。

int x = (1 << 32) - 1;         // 0
int y = (int)((1L << 32) - 1); // -1

配列

Java の配列は固定長ですが、参照型です。 また、型は長さ情報を持ちません。

int[] a = { 0, 1, 2 };

// class [I
System.out.println(a.getClass());

// 長さは違うが代入できる
a = new int[]{0, 1};

宣言における [] の位置は柔軟です。

// どちらでも良い
int[] d;
int e[];

// 全て同じ型 (intの2次元配列)
int[][] a;
int[] b[];
int c[][];

宣言時の初期化は3通りの方法があります。

int[] a = new int[10];
int[] b = new int[]{4, 5};
int[] c = {0, 1};

サイズと初期値を同時に指定することはできません。

// エラー
var b = new int[2]{4, 5};

代入では {0, 1} のような書き方ができません。

a = new int[10];
b = new int[]{4, 5};

// エラー
// c = {0, 1, 2};

長さは動的に決められます。

int n = 3;
int[] c = new int[n];

多次元配列において、より左側に書かれる括弧が外側です。

var a = new int[2][3];
System.out.println(a.length); // 2

初期化時には、最も外側の要素数を確定させなくてはいけません。

// 一番外側だけ確定していれば良い
a = new int[2][];

// エラー
// b = new int[][];

arraycopy で C の memcpy 相当の操作ができます。

System.arraycopy(b, 0, c, 0, b.length);
