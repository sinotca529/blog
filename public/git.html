Git 簡易リファレンス

.gitignore の書き方

# 任意の場所にある
# foo というファイル/ディレクトリを無視
foo

# 任意の場所にある
# bar というディレクトリを無視
bar/

# プロジェクトルートから /baz の位置にある
# ファイル/ディレクトリを無視
/baz

# プロジェクトルートから /qux の位置にある
# ディレクトリを無視
/qux/

# 任意の場所にある .o ファイル/ディレクトリを無視
*.o

git worktree

ブランチ切替のために毎回 stash するのは大変面倒です。 git worktree はこの煩わしさを解消します。

git worktree は、ブランチをディレクトリ下に展開するコマンドです。 展開されたディレクトリ内で作業し、要が済んだらワークツリーを削除してもとの作業に戻ります。

# ワークツリーを作成する。
# `作業ディレクトリ` はデフォルトでブランチ名が使われます。
# 現在のブランチは指定できないようです。
git worktree add <作業ディレクトリ> <ブランチ名>

# 作業ディレクトリを削除する
git worktree remove <作業ディレクトリ>

# 作業ディレクトリ一覧
git worktree list
