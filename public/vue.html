Vue

vue コマンドのインストール

vue コマンドをインストールします。

npm install -g @vue/cli

プロジェクトの作成

vue コマンドを使ってプロジェクトを作ります。 Vue 2 と Vue 3 のどちらにするか聞かれたら 3 を選びます。

vue create vue-test
cd vue-test

作成されたプロジェクトは次のようになっています。

.
├── babel.config.js
├── jsconfig.json
├── node_modules
│  ├── @achrinza
│  ├── ...
│  └── yorkie
├── package-lock.json
├── package.json
├── public                // 静的ファイルを格納
│  ├── favicon.ico
│  └── index.html
├── README.md
├── src                   // ソースコードを格納
│  ├── App.vue            //   アプリのルートコンポーネント
│  ├── assets             //
│  │  └── logo.png        //
│  ├── components         //   再利用可能なコンポーネントを格納
│  │  └── HelloWorld.vue  //
│  └── main.js            //   エントリポイント
└── vue.config.js         // ビルド設定などを書く

実行は次のようにします。

npm run serve

コンポーネントの基本

ここでは、 App.vue を書き換えて、テンプレート構文, データバインディング, イベントについて学びます。

template

App.vue の内容を削除して、はじめに <template> を書きます。 ここには HTML のテンプレートを書きます。

<template>
  <div id="app">
    <h1>{{ title }}</h1>
    <p>{{ message }}</p>
    <input v-model="inputText" placeholder="テキストを入力">
    <button v-bind:disabled="inputText.length === 0" @click="greet">挨拶する</button>
    <p>クリック回数: {{ clickCount }}</p>
  </div>
</template>

静的サイトジェネレータでおなじみの Mustache に似ています。

v-model, v-bind や @click はディレクティブといいます。 上の例におけるそれぞれの意味は次のとおりです。

  • v-model="inputText" : 入力フィールドとデータプロパティ inputText を双方向にバインドする
  • v-bind:disabled="inputText.length === 0 : 条件が満たされるときだけ disabled をつける
  • @click="greet" : クリック時に greet メソッドを呼び出す

script

続けて <script> を書きます。 ここにはテンプレートに与えるパラメタを記述します。

<script>
export default {
  name: 'App',
  data() {
    return {
      title: 'Vue.js チュートリアル',
      message: 'テキストを入力してボタンをクリック！',
      inputText: '',
      clickCount: 0
    }
  },
  methods: {
    greet() {
      this.message = `こんにちは、${this.inputText}さん！`;
      this.clickCount++;
    }
  }
}
</script>

name は Vue コンポーネントの識別子です。

style

最後に <style> を書きます。

<style>
#app {
  font-family: Arial, sans-serif;
  text-align: center;
  margin-top: 60px;
}
</style>

複数のコンポーネントを使う

<p>{{ message }}</p> の部分をコンポーネントとして切り出しましょう。

新規コンポーネントの作成

MsgPara.vue を作成します。

<template>
  <p>{{ message }}</p>
</template>

<script>
export default { props: ['message'] }
</script>

<style scoped>
p { color: blue; }
</style>

<style scoped> と書くと、そのスタイルはコンポーネント内にしか適用されなくなります。

props は親コンポーネントから子コンポーネントに値を渡すためのものです。 子コンポーネントは値の読み取りはできますが、書き込みはできません。

コンポーネントの利用

App.vue を次のように書き換えます。

<template>
  <div id="app">
    <h1>{{ title }}</h1>
    <MsgPara :message="message" />
    ...
  </div>
</template>

<script>
import MsgPara from './components/MsgPara.vue'

export default {
  name: 'App',
  components: {
    MsgPara
  },
  ...
}
</script>
...
