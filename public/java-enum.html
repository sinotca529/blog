(Java) enum

Java の enum は java.lang.Enum を継承する final クラスとしてコンパイルされます。 列挙子variantは public static final なインスタンスとして表現されます。 あくまでクラスなので、コンストラクタ, フィールド, メソッドを定義できます。

列挙子のインスタンスが作成されるのはクラスのロード時です。 enum を初めて使うときに全バリアントのコンストラクタが呼ばれます。

enum Signal {
  RED("red"), GREEN("green"), YELLOW("yellow");

  private final String color;

  private Signal(String color) {
    System.out.println(color);
    this.color = color;
  }
}

public class Main {
  public static void main(String args[]) {
    // 列挙子が作成され、 red green yellow と表示される。
    var s = Signal.RED;
  }
}

これは enum 特有の挙動ではなく、静的メンバの挙動です。 通常のクラスの静的メンバについても同じ現象が起こります。

class A { A(String s) { System.out.println(s); } }

class B {
  static final A a1 = new A("1");
  static final A a2 = new A("2");
}

public class Main {
  public static void main(String args[]) {
    // B.a1, B.a2 が作成され、 1 2 と表示される。
    var s = B.a1;
  }
}
