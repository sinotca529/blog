vim 簡易リファレンス

ファイルの暗号化

# 開くときに暗号化
vim -x file

# パスワード変更 (新パスワードを未入力にすると暗号化解除)
vim +X file
  • 編集中に暗号化を有効にするには、 :X を押します。
  • 暗号化方式を指定するには :set cryptmethod を使用します。

エンコードを指定して開きなおす

:e ++enc=sjis

バイナリファイルを開く

xxd コマンドを利用する。

  1. vim -b file で開く
  2. :%!xxd でバイナリ表示
  3. 編集する
  4. :%!xxd -r で元に戻す

タブ文字を表示する

:set list listchars=tab:>-
