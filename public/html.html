dataset プロパティ

要素に data-* 属性を持たせて、それを参照する機能。
例えば、ページごとに実行する javascript を変えるのに使える。

window.addEventListener("DOMContentLoaded", () => {
  switch (document.body.dataset.page) {
    case "tag":
      tagMain();
      break;
    case "index":
      indexMain();
      break;
  }
});

<body data-page="index">
  ...
</body>

<body data-page="tag">
  ...
</body>

スクリプトを起動するタイミング

  • onLoad: すべてのリソースが読み込まれたら発火
  • DOMContentLoaded : HTMLの解釈が終了したら発火

要素をマウスでグリグリ動かす

出典

ドラッグ中かどうかのフラグ管理なしで実現できる。

<img id="img" src="https://js.cx/clipart/ball.svg" width="40" height="40">
<script>
img.onpointermove = function(e) {
    if(e.buttons){
        this.style.left     = `${this.offsetLeft + e.movementX}px`;
        this.style.top      = `${this.offsetTop + e.movementY}px`;
        this.style.position = 'absolute';
        this.draggable      = false;
        this.setPointerCapture(e.pointerId);
    }
}
</script>
