ユーザー環境変数を設定する

ユーザー環境変数の設定に管理者権限は不要です。 コントロールパネルから「ユーザーアカウント > ユーザーアカウント > 環境変数の変更」と進むことで設定画面を開けます。 あるいは、 Win + R のダイアログに次を入力することでも設定画面を開けます。

rundll32 sysdm.cpl,EditEnvironmentVariables

ウィンドウの角丸をやめる

Windows Registry Editor Version 5.00

[HKEY_CURRENT_USER\Software\Microsoft\Windows\DWM]
"ForceEffectMode"=dword:00000002

Git Bash で対話的ソフトを使う

Python インタプリタなどを Git Bash でそのまま動かそうとするとハングする。 これは、 pythone.exe と Git Bash に互換性がないから。 winpty を嚙ませると、このあたりの問題が解消する。

ただし、16色しか使えない。

winpty python

のように使う。
