Java のパッケージ

宣言

ファイルの冒頭で package foo.bar; のように宣言します。 パッケージ名はディレクトリ構造と同じでなくてはいけません。

// src/foo/bar/A.java
package foo.bar;

public class A {
  public static void main(String[] args) {
    System.out.println("Hello");
  }
}

複数のファイルが同じパッケージに所属することもあります。

.
└── src
   └── foo
      ├── bar
      │  ├── A.java  // package foo.bar
      │  └── B.java  // package foo.bar
      └── baz
         └── C.java  // package foo.baz

Java のクラスは必ずパッケージに属する必要があります。 パッケージ宣言がないファイルに書かれたクラスは、デフォルトパッケージに所属します。

インポート

Java で import できるのは定数とクラスのみです。

import static java.lang.Math.PI;
import java.util.ArrayList;

Rust の use のようにパスを読み込むことはできません。 必ずクラス名まで書く必要があります。

// エラー
// import java.util

コンパイル

依存先のファイルは自動でコンパイルされます。

javac \
  # 依存先 java ファイルの探索先
  --source-path src \
  # コンパイル対象
  src/foo/bar/A.java
