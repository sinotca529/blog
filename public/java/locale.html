(Java) ロケール

国際化対応 (i18n) には java.util.Locale とその関連クラスを利用します。

// システム規定のロケールを取得
Locale l = Locale.getDefault();
System.out.println(l); // en_US

ロケールの作成

ロケールを作成するにはコンストラクタを利用します。 コンストラクタは3つあります。 language には言語コードを、 country には国コードを指定します。 variant はより細かな分類が必要な際に利用します。

  • Locale(String language)
  • Locale(String language, String country)
  • Locale(String language, String country, String variant)

これらコンストラクタは引数の整合性をチェックしません。 メジャーなロケールであれば定数を利用できます。

Locale l = Locale.US; // en_US 相当

他にも、IETF言語タグも利用できます。

Locale l = Locale.forLanguageTag("en-US")

ビルダーパターンによる構築も可能です。

Builder b = new Locale.Builder();
Locale l = b
    .setLanguage("en")
    .setRegion("US")
    .build();
