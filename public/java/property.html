(Java) プロパティファイル

java.util.Properties クラスを利用します。

プロパティの読み込み

Properties p = new Properties();
p.load(new FileReader("settings.properties"))
for (Object key: p.keySet()) System.out.println(p.get(key));

キーを文字列型で指定できるならば、 get の代わりに getProperty を利用します。 get は引数も戻り値も Object ですが、 getProperty はともに String です。

var name = p.getProperty("name");

プロパティファイルの記述

プロパティファイルに利用できる文字コードは ISO-8859-1 (Latin-1) もしくは UTF-8 です。 ただし、 UTF-8 が利用できるのは Java SE 9 以降です。 それ以前、日本語を扱いたい場合は native2ascii コマンドを用いて ISO-8859-1 への変換を行っていました。

# 変換前
name=太郎

# 変換後
name=\u592a\u90ce

言語の切り替え

利用者の環境に応じて読み込むプロパティファイルを変更するには java.util.ResourceBundle を利用します。

// デフォルトのロケールが en_US の場合 settings_en_US.properties を読み込む。
// 存在しない場合は settings.properties を読み込む。
ResourceBundle rb = ResourceBundle.getBundle("settings");
System.out.println(rg.getString("name"));

getBundle はファクトリメソッドです。
