ラポール

ラポールの構築方法

相手と自分が「同じ」であると思わせることが重要。

ベーシング : 話し方を相手に揃える ミラーリング : 表情・姿勢を揃える バックトラッキング : 鸚鵡返しする。

自己紹介などにより、共通部分を見つける活動もこの一種と言える。

ラポールの破壊

「同じ」であると思わせるのは難しい。 本心でないことを言っていたりするとバレる。

伝え方

相手の理解 → 調査 → ストーリー作成 → 資料作成 → 対話

人の累計 : 感情表現を抑える アナリティカル ↑ ドライビング 意見を聞く ← → 意見を主張する エミアブル ↓ エクスプレッシブ 感情を表す

ドライビング

  • 資料 + 口頭説明

エクスプレッシブ

  • スペシャル感を出す

アナリティカル

  • 打ち合わせ前に資料送付

エミアブル

  • 関係者の同意事項を

空・雨・傘

空 (事実) → 空を見上げると曇っている 雨 (解釈) → 今日は雨が降りそうだ 傘 (行動) → 傘を持って出かけよう

会話の目的は相手にアクションを取ってもらうことであって、事実の伝達や、相手に判断を委ねることではない。 事実の伝達だけでは不十分。 要求の明示が必要。

案が悩ましいケース : 3 つ出すのが良い。 比較の軸が明確になる。 複数から選ぶことで、判断している空気が出せる。 (大体はエイヤで困らない気もするが、判断してる感があると説得力がある)

  • 説明資料
  • 補足資料
