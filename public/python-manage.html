Python のパッケージ管理

Python のパッケージ管理は大変ですから、様々な管理ツールが開発されています。 そうしたツールの使い方をまとめます。

ツール名uv
特定の python をいれるuv python install 3.11.7
導入済み python の確認uv python list
初期化uv init my-proyect
依存追加uv add numpy
依存削除uv remove numpy
ロックファイル生成uv lock
環境の再現uv sync
実行uv run main.py
