Linux と Windows を並用すると時計がずれる

Linux を起動したあとのマシンで Windows を起動すると、時計がずれていることがある。 この問題は、 Linux と Windows でハードウェアクロックの使い方が違うことに起因する。

OSハードウェアクロックの使い方
Linux世界標準時をセット
Windowsローカルの時刻をセット

例えば日本だと9時間のズレが生じることになる。

この問題を解消するには、 Linux か Windows の設定を変えてやれば良い。 簡単なのは Linux で、次のコマンドを実行すれば良い。

sudo timedatectl set-local-rtc 1
