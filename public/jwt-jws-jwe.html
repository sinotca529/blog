JWT/JWS/JWE (アクセストークン)

JWT は、ステートレスな Web アプリでアクセストークンのやり取りに使うフォーマットです。 JWS は JWT に署名を付けたもので、 JWE は JWT を暗号化したものです。

Cookie を使いたくないときや、ステートレスなサーバを作りたいときなどに重宝します。

JWT (JSON Web Token)

   Header                    Claim
  ┌────────────────┐        ┌─────────────────────┐
  │ {              │        │ {                   │
  │   "typ": "JWT" │        │   "sub": "user01",  │
  │ }              │        │   "exp": 1735359783 │
  └────────────────┘        │ }                   │
          │                 └─────────────────────┘
          │                            │
          │ Base64                     │ Base64
          v                            v
   eyJ0eXAiOiJKV1QifQ    .  eyJzdWIiOiJ1c2VyMDEiLCJleHAiOjE3MzUzNTk3ODN9

ヘッダとクレームをそれぞれ Base64 に変換して、 . で繋げます。 作った文字列は HTTP の Authorization ヘッダにもたせます。

JWS (JSON Web Signiture)

JWS の場合は、ここに署名が加わります。

   Header                                   Claim
  ┌──────────────────┐                     ┌─────────────────────┐
  │ {                │                     │ {                   │
  │   "typ": "JWT",  │                     │   "sub": "user01",  │
  │   "alg": "HS256" │                     │   "exp": 1735359783 │
  │ }                │                     │ }                   │
  └──────────────────┘                     └─────────────────────┘
          │                                        │
          │ Base64                                 │ Base64
          v                                        v
   eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9  .   eyJzdWIiOiJ1c2VyMDEiLCJleHAiOjE3MzUzNTk3ODN9
  ────────────────────────────────────────────────────────────────────────────────────────
           │                         │
           │                         └─────────────────────────────────────────────────────> xhlm34-rBtgLBGdM9edJrwZ-mk2sXi39rmtNS8cj1eY
           │                            Sign & Base64                                       ─────────────────────────────────────────────
           │                                                                                        │
           v                                                                                        v
